🚗 CarsCave – Car Showcase Website

CarsCave is a simple and interactive car showcase website built using HTML, CSS, and Bootstrap. This project demonstrates basic frontend development concepts including navigation, dropdown menus, image carousel, and a login form.

🌟 Features
🚘 Car brand selection (Rolls Royce, Lamborghini, Mercedes, Toyota)
📂 Category filtering (Racing, Hybrid, SUV, etc.)
💰 Pricing range selection
🎞️ Image carousel showcasing different cars
🧭 Responsive navigation bar with dropdown menus
🔐 Login form with email and password input
🎨 Clean UI using Bootstrap components

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5

📂 Project Structure
├── index.html        # Main webpage
├── style.css         # Custom styles
├── images/           # Car images and assets
├── page.php          # Login redirect page (optional backend)

🎯 Purpose

This project is created as a practice project to improve:

Frontend development skills
Bootstrap usage
UI layout and structure design

⚙️ How to Run
Download or clone this repository
Open index.html in your browser
Ensure all images and CSS files are properly linked
